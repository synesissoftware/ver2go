## Installation

Install via `go get`, as in:

```bash
go get "github.com/synesissoftware/ver2go"
```

and then import as:

```Go
import ver2go "github.com/synesissoftware/ver2go"
```

or, simply, as:

```Go
import "github.com/synesissoftware/ver2go"
```
